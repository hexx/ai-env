`pi-projects.ts` ファイル内の `generatePiResumeFunc` 関数に対して、以下の2点の仕様変更を行ってください。

### 変更要件
1. **コマンドの変更**: コンテナ内で実行するコマンドを `pi --resume` ではなく `pi --session` に変更してください。
2. **ディレクトリの自動認識**: `pi-resume` コマンド実行時に引数が省略された場合、カレントディレクトリ名（`basename "$PWD"`）をプロジェクト名として自動的に使用するように変更してください。

### 修正イメージ
修正後の `generatePiResumeFunc` は以下のようになる想定です。これに合わせてコードを修正してください。

```typescript
const generatePiResumeFunc = (
  piProjects: Record<string, string>,
): string => {
  const cases = Object.entries(piProjects)
    .map(
      ([project, sessionId]) =>
        `    ${project}) pi --session ${sessionId} ;;`, // <-- --session に変更
    )
    .join("\n");
  const available = Object.keys(piProjects).join(" ");
  return [
    "pi-resume() {",
    '  local project="${1:-$(basename "$PWD")}"', // <-- 引数がない場合はカレントディレクトリ名を使用
    '  case "$project" in',
    cases,
    '    *) echo "Unknown project: $project" >&2',
    `       echo "Available: ${available}" >&2`,
    "       return 1 ;;",
    "  esac",
    "}",
  ].join("\n");
};
