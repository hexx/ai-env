コードのリファクタリングは素晴らしいですが、`child_process.spawnSync` の引数の渡し方にバグがあります。

`spawnSync` の引数配列において、`-e HOGE=PIYO` や `-v /path:/workspace` のようにスペースを含む1つの文字列を渡すと、コマンドライン引数として正しく解釈されません。

`index.ts` の `envArgs` と `volumeArgs` の定義を、`-e` や `-v` ではなく、`--env=...` および `--volume=...` のように `=` で繋ぐ形式に修正してください。

例:
`-e HERDR_PANE_ID=${herdrPaneId}` -> `--env=HERDR_PANE_ID=${herdrPaneId}`
`-v ${cwd}:/workspace` -> `--volume=${cwd}:/workspace`
